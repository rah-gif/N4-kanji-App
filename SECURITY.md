# Security Policy

## Supported Versions

Below are the versions of N4 Kanji App that receive security updates:

| Version | Status         | Description                                              |
| ------- | -------------- | -------------------------------------------------------- |
| v1.0.0  | ✅ Supported   | **Current Stable Release** - AI-Powered N4 Kanji Master. |
| < 1.0   | ❌ End of Life | Beta versions.                                           |

## Reporting a Vulnerability

We prioritize the security of our AI integrations. If you discover a vulnerability, please follow these steps:

1.  **Do not** create a public GitHub issue.
2.  Please report vulnerabilities via the "Private vulnerability reporting" feature on this repository (if enabled) or contact the maintainer directly.
3.  We aim to acknowledge reports within 48 hours.

Thank you for helping keep the N4 Kanji App secure!
